carouselImg.js

This script will allow you to create interactive image carousel.

Demo

Example

<section id="carouselAnim"></section>
<article id="carousel">
    <figure>
        <a href="#lake" title="Lake">
            <img src="img/nature001.jpg" alt="Sunset at lake" />
        </a>
        <figcaption>&copy; Spudulike</figcaption>
    </figure>
    <figure>
        <a href="#mountains" title="Mountains">
            <img src="img/nature002.jpg" alt="Mountain house" />
        </a>
        <figcaption>&copy; Dariusz Marcinek</figcaption>
    </figure>
    <figure>
        <a href="#forest" title="Forest">
            <img src="img/nature003.jpg" alt="Forest at night" />
        </a>
        <figcaption>&copy; Dragomir Uzunov</figcaption>
    </figure>
</article>
<script>
    new Carousel('carousel',{
        containerToRenderId:'carouselAnim',
        bgPosition: 'center',
        displayTime: 10,
        drawSkitHeader: true,
        drawSkitText: true,
        drawSkitButton: true,
        drawPausePlayButton: true
    });
</script>

Syntax

new Carousel('id_of_element', [options]);

Options

activeClassName
Sets rendered menu active state class name attribute, defaults to 'selected'.
behaviour
Sets menu behaviour during click, 'change' or 'slide' , defaults to 'change'.
behaviourDuration
Sets rendered elements behaviour duration in seconds, defaults to 10.
bgPosition
Sets background image position attribute, '50% 50%' or 'center' , defaults to 'left center'.
boxClassName
Sets rendered container class name attribute, defaults to 'boxCarousel'.
buttonClassName
Sets rendered button class name attribute, defaults to 'boxCarouselButton'.
buttonNextArrowClassName
Sets rendered arrow class name attribute, defaults to 'boxCarouselArrowNext'.
buttonPreviousArrowClassName
Sets rendered arrow class name attribute, defaults to 'boxCarouselArrowPrevious'.
buttonPauseClassName
Sets rendered pause button class name attribute, defaults to 'boxCarouselButtonPlay'.
buttonPlayClassName
Sets rendered play button class name attribute, defaults to 'boxCarouselButtonPause'.
buttonPauseText
Sets text node for rendered content <button>, defaults to 'pause'.
buttonPlayText
Sets text node for rendered content <button>, defaults to 'play'.
contentClassName
Sets rendered div class name attribute, defaults to 'boxCarouselContent'.
containerToRenderId
Sets id attribute of element where carousel will be rendered, defaults to 'carouselAnim'.
drawNav
Disable/enable naviagtion rendering - boolean, defaults to true.
drawNavArrows
Disable/enable naviagtion arrows rendering - boolean, defaults to true.
drawSkitHeader
Disable/enable img title tag rendering - boolean, defaults to true.
drawSkitSubHeader
Disable/enable img alt tag rendering - boolean, defaults to true.
drawSkitText
Disable/enable img data-long-desc tag rendering - boolean, defaults to true.
drawSkitButton
Disable/enable img data-skit-button-text tag rendering - boolean, defaults to true.
drawPausePlayButton
Disable/enable pause button rendering - boolean, defaults to true.
menuClassName
Sets rendered menu class name attribute, defaults to 'boxCarouselNav'.
navigationTextAttr
Sets rendered text node for navigation <li> element'alt'.
skitButtonTextAttr
Sets rendered text node for content <a> element 'alt'.
skitHeaderAttr
Sets rendered text node for content <h1> element 'title'.
skitSubHeaderAttr
Sets rendered text node for content <h2> element 'title'.

intervalInputs.js

This script will allow you to create one input out of many without putting mask.

Demo

Type your

Example

<form id="interval">
    <fieldset class="interval">
        <legend>Type your </legend>
        <label>MAC address:</label>
        <input name="mac1" value="" maxlength="2" />
        <input name="mac2" value="" maxlength="2" />
        <input name="mac3" value="" maxlength="2" />
        <input name="mac4" value="" maxlength="2" />
        <input name="mac5" value="" maxlength="2" />
        <input name="mac6" value="" maxlength="2" />
    </fieldset>
</form>        
<script>
    new IntervalInputs('.interval',{
        intervalChar: '-',
        intervalWidth: 1,
        intervalUnit: 'em',
        scaleAttribute: 'maxlength',
        autoFocusChange: true
    });
</script>

Syntax

new IntervalInputs('.css_selector #sizzle', [options]);

Options

autoFocusChange
Sets inputs focus behaviour, defaults to true.
intervalDivClassName
Sets rendered interval div class name attribute, defaults to 'intervalDiv'.
intervalWidth
Sets inputs css width property, defaults to 1.
intervalUnit
Sets inputs css width property unit, defaults to 'em'.
scaleAttribute
Sets inputs element scale attribute, defaults to 'maxlength'.
